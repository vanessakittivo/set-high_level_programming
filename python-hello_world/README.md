# Python - Hello, World

This project contains introductory exercises in Python programming.

## Learning Objectives

At the end of this project, I am able to:

- Run Python scripts.
- Execute Python code from the command line.
- Print strings, integers, and floats.
- Use Python variables.
- Work with string slicing and concatenation.
- Use formatted strings (f-strings).
- Understand basic Python syntax.
- Detect a cycle in a singly linked list using C.

## Files

| File | Description |
|------|-------------|
| `0-run` | Shell script that runs a Python script. |
| `1-run_inline` | Shell script that runs inline Python code. |
| `2-print.py` | Prints a simple string. |
| `3-print_number.py` | Prints an integer using an f-string. |
| `4-print_float.py` | Prints a float with two decimal places. |
| `5-print_string.py` | Prints a string three times and its first nine characters. |
| `6-concat.py` | Concatenates two strings. |
| `7-edges.py` | Demonstrates string slicing. |
| `8-concat_edges.py` | Creates a new sentence using string slicing. |
| `9-easter_egg.py` | Prints "The Zen of Python". |
| `10-check_cycle.c` | Detects whether a singly linked list contains a cycle. |
| `lists.h` | Header file for the linked list project. |

## Requirements

- Ubuntu 20.04 LTS
- Python 3
- GCC
- Betty coding style for C
