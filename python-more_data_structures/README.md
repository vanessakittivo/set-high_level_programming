# Python - More Data Structures


## Learning Objectives

By completing this project, I learned how to:

- Work with sets and perform set operations
- Manipulate dictionaries efficiently
- Use list and dictionary comprehensions
- Apply lambda functions with `map()`
- Convert Roman numerals to integers
- Calculate weighted averages
- Delete dictionary entries safely
- Use Python's built-in functions effectively
- Explore Python internals using the CPython C API

---

## Project Structure

| Task | File | Description |

| 0 | `0-square_matrix_simple.py` | Returns a new matrix with all values squared |

| 9 | `9-multiply_by_2.py` | Returns a dictionary with all values multiplied by 2 |

| 11 | `11-multiply_list_map.py` | Multiplies list elements using `map()` |
