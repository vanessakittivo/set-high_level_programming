# Python - Import & Modules

This project is part of the **Elmwood Python Programming** curriculum.

## Project Overview

This project introduces Python modules, importing functions, command-line arguments, and basic scripting. It also covers bytecode interpretation and creating reusable Python functions.

## Learning Objectives

By completing this project, I learned how to:

- Import functions from another file.
- Use Python modules.
- Create executable Python scripts.
- Use command-line arguments.
- Work with `argv`.
- Create and use Python functions.
- Read and reproduce Python bytecode.
- Follow Python coding style (PEP 8 / pycodestyle).

## Project Structure

| File | Description |
|------|-------------|
| `0-add.py` | Imports the `add` function and prints the result of adding two numbers. |
| `1-calculation.py` | Imports calculator functions and performs arithmetic operations. |
| `2-args.py` | Prints the number of command-line arguments and their values. |
| `3-infinite_add.py` | Adds all command-line arguments. |
| `4-hidden_discovery.py` | Prints all names defined by a compiled module. |
| `5-variable_load.py` | Imports and prints the value of a variable. |
| `100-my_calculator.py` | Simple command-line calculator. |
| `101-easy_print.py` | Prints `#pythoniscool` without using `print()`. |
| `102-magic_calculation.py` | Recreates a Python function from bytecode. |
| `103-fast_alphabet.py` | Prints the uppercase alphabet without loops. |
