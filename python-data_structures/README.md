# Python - Data Structures

This project is part of the **Elmwood / ALX Higher Level Programming** curriculum.

## Description

This project introduces Python data structures and teaches how to work with:

- Lists
- Tuples
- Nested lists
- List comprehensions
- Matrix operations
- Strings
- Functions
- Basic algorithms
- C programming interview questions

The goal is to understand how Python stores and manipulates collections of data while writing clean, Pythonic code that follows the **Pycodestyle (PEP 8)** style guide.

---

## Learning Objectives

By completing this project, I learned how to:

- Use Python lists effectively
- Access and modify list elements
- Slice lists and strings
- Work with tuples
- Return multiple values using tuples
- Build and manipulate matrices
- Write reusable Python functions
- Find maximum values manually
- Delete and modify list elements
- Swap variable values
- Solve linked list interview problems in C
- Explore Python internals using the CPython C API

---

## Project Structure

| File | Description |
|------|-------------|
| `0-print_list_integer.py` | Print all integers in a list |
| `1-element_at.py` | Retrieve an element from a list |
| `2-replace_in_list.py` | Replace an element in a list |
| `3-print_reversed_list_integer.py` | Print a list in reverse |
| `4-new_in_list.py` | Replace an element without modifying the original list |
| `5-no_c.py` | Remove all `c` and `C` characters from a string |
| `6-print_matrix_integer.py` | Print a matrix of integers |
| `7-add_tuple.py` | Add two tuples |
| `8-multiple_returns.py` | Return the length and first character of a string |
| `9-max_integer.py` | Find the largest integer in a list |
| `10-divisible_by_2.py` | Determine multiples of two |
| `11-delete_at.py` | Delete an item at a given position |
| `12-switch.py` | Swap two variables |
| `13-is_palindrome.c` | Check whether a linked list is a palindrome |
| `100-print_python_list_info.c` | Print information about Python lists using the CPython API |
