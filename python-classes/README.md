# Python - Classes and Objects

This project introduces Object-Oriented Programming (OOP) in Python.

## Learning Objectives

By completing this project, I learned how to:

- Create classes and objects
- Define constructors using `__init__`
- Work with private instance attributes
- Validate user input
- Use properties, getters, and setters
- Create instance methods
- Calculate object properties
- Print custom object representations
- Compare objects using magic methods
- Build a sorted singly linked list
- Apply Python's object-oriented programming principles

## Requirements

- Ubuntu 20.04 LTS
- Python 3.8.5
- All files executable
- All files begin with:

```python
#!/usr/bin/python3
```

- Code follows **pycodestyle (2.8.\*)**
- No unnecessary comments or docstrings
- README included
