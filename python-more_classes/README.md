# Python - More Classes

This project contains solutions for the Python More Classes tasks.

## Learning Objectives

- Classes and objects
- Attributes
- Properties and setters
- Instance methods
- Class attributes
- Static methods
- Class methods
- Object representation
- Instance deletion
- Backtracking algorithms
- Python style using pycodestyle

## Files

| File | Description |
