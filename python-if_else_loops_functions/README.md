# Python - If/Else, Loops, Functions

This repository contains my solutions for the **Python - If/Else, Loops, Functions** project in the **set-high_level_programming** repository.

## 📚 Project Overview

This project introduces the fundamentals of Python programming, including:

- If, elif, and else statements
- Comparison and logical operators
- For loops
- Functions
- String manipulation
- Number manipulation
- ASCII character operations
- Basic algorithms
- Introductory linked list programming in C

---

## 🎯 Learning Objectives

By completing this project, I learned how to:

- Write clean Python scripts that follow PEP 8 standards.
- Use conditional statements to make decisions.
- Iterate through sequences using loops.
- Create and use functions.
- Work with strings and integers.
- Manipulate characters using their ASCII values.
- Build simple algorithms.
- Implement basic linked list operations in C.

---

## 📁 Project Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, negative, or zero. |
| `1-last_digit.py` | Prints the last digit of a random number with the appropriate message. |
| `2-print_alphabet.py` | Prints the lowercase alphabet. |
| `3-print_alphabt.py` | Prints the lowercase alphabet except `q` and `e`. |
| `4-print_hexa.py` | Prints numbers from 0 to 98 in decimal and hexadecimal. |
| `5-print_comb2.py` | Prints numbers from 00 to 99. |
| `6-print_comb3.py` | Prints all possible unique combinations of two digits. |
| `7-islower.py` | Checks whether a character is lowercase. |
| `8-uppercase.py` | Prints a string in uppercase. |
| `9-print_last_digit.py` | Prints and returns the last digit of a number. |
| `10-add.py` | Adds two integers and returns the result. |
| `11-pow.py` | Computes a number raised to a given power. |
| `12-fizzbuzz.py` | Prints the FizzBuzz sequence from 1 to 100. |
| `13-insert_number.c` | Inserts a node into a sorted singly linked list. |
